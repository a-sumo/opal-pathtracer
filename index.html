Opal Path Tracer
spectral · photonic crystal · webgl

Precious opal gets its play-of-color from Bragg diffraction: light bouncing off ordered arrays of silica nanospheres packed into a photonic crystal. This is a physically-based spectral path tracer that renders that effect from first principles. Every camera ray carries one wavelength, scatters off randomly-oriented crystal grains baked into a Voronoi volume, and accumulates into a CIE XYZ buffer, so the color you see is decided entirely by which wavelengths make it back out of the stone.

How it works
1
Volume bake opal-volume-baker.js
A CPU worker prepares a compact domain map for the stone, including local grain orientations and a few small textures that the shader can read quickly.
2
Spectral path trace pathtracer.html
A fullscreen fragment shader traces wavelength samples through the opal body and lets the local lattice geometry decide which colors flash back to the camera.
3
Display
Spectral samples are accumulated first, then converted to display color at the end so the image stays tied to the optical path.
